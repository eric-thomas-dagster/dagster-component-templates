# DataEnricherComponent

> **üîë API key required.** This component calls an LLM provider. Set `OPENAI_API_KEY` for OpenAI (default), or configure an alternate provider (Anthropic / Azure OpenAI / Ollama / etc.) via the component's `provider`, `model`, and `api_key_env_var` fields. See the schema for the exact field names this component exposes.

LLM-driven row enrichment ‚Äî for each row in an upstream DataFrame, send selected
context columns to an LLM and parse structured fields back. Useful for tagging,
sentiment, classification, summarization, entity extraction.

## Dependencies

- `openai` (or your LLM provider's client)
- `pandas`
