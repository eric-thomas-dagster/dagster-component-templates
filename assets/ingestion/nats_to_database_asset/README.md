# NATS to Database Asset

Fetches messages from a NATS subject or JetStream consumer and writes them to a database table via SQLAlchemy. Supports both core NATS (ephemeral) and JetStream (durable, acknowledged).

## Required packages

```
nats-py>=2.6.0
pandas>=1.3.0
sqlalchemy>=2.0.0
```

## Asset Dependencies & Lineage

This component supports a `deps` field for declaring upstream Dagster asset dependencies:

```yaml
attributes:
  # ... other fields ...
  deps:
    - raw_orders              # simple asset key
    - raw/schema/orders       # asset key with path prefix
```

`deps` draws lineage edges in the Dagster asset graph without loading data at runtime. Use it to express that this asset depends on upstream tables or assets produced by other components.

Dependencies can also be wired externally via `map_resolved_asset_specs()` in `definitions.py` ‚Äî the same approach used by [Dagster Designer](https://github.com/eric-thomas-dagster/dagster_designer).
