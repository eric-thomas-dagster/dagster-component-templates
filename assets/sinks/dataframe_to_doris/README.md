# dataframe_to_doris

## Purpose

Bulk-ingest a Pandas DataFrame into an [Apache Doris](https://doris.apache.org/) table via Doris's **Stream Load** HTTP API ‚Äî the production high-throughput ingestion path on Doris (similar to Snowflake's COPY INTO). PUTs the serialized DataFrame to `http://{fe}:{http_port}/api/{db}/{table}/_stream_load`. Typical throughput: 100k‚Äì1M rows/sec per FE per table.

For tens of rows, `dataframe_to_table` against `doris_resource.connection_string` works fine. For any real volume, Stream Load is the right path.

```yaml
  password_env_var: DORIS_PASSWORD
  mode: append
  format: csv
```

## Modes

- `mode: append` (default) ‚Äî Stream Load appends rows.
- `mode: replace` ‚Äî pre-DELETE rows matching `replace_predicate` (a SQL WHERE clause), then Stream Load. **Requires** a Unique Key or Aggregate table model with delete-sign support; Duplicate Key tables will fail.
