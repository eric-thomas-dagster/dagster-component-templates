# DataFrame ‚Üí Dynatrace Events

Push DataFrame rows as Dynatrace events via the v2 Events API. Use for batch-job markers (deployments, ETL completions, anomalies) that should annotate Dynatrace's timeline.

## Event types

- CUSTOM_INFO ‚Äî generic info events
- CUSTOM_DEPLOYMENT ‚Äî deployment markers (great for ETL job completion)
- CUSTOM_ANNOTATION ‚Äî annotations on existing events
- AVAILABILITY_EVENT, ERROR_EVENT, PERFORMANCE_EVENT, RESOURCE_CONTENTION_EVENT ‚Äî synthetic events that drive Dynatrace problem detection
