# SampleComponent

Sample rows from a DataFrame. Pick a `method`:

| Method | What it does | Driven by |
|---|---|---|
| `random` (default) | `pd.DataFrame.sample(...)` | `sample_size` or `frac`, plus `random_state` / `replace` / `weights` |
| `head` | First N rows | `sample_size` |
| `tail` | Last N rows | `sample_size` |
| `every_nth` | Rows at positions 0, N, 2N, ... | `sample_size` (the step) |
| `skip_head` | Drop the first N rows, return the rest | `sample_size` |

## Use Cases

- Create smaller representative subsets for exploratory analysis (`random`)
- Bootstrap resampling for statistical estimates (`random` + `replace=true`)
- Smoke-test against the first / last few rows of a feed (`head` / `tail`)
- Subsample evenly across an ordered series (`every_nth`)
- Drop a header preamble before processing (`skip_head`)
- Migrating an Alteryx **Sample** tool — the `method` field maps 1:1 to its options

## Example YAML

### Random (default)

```yaml
type: dagster_component_templates.SampleComponent
attributes:
  asset_name: transactions_sample
  upstream_asset_key: raw_transactions
  method: random           # optional — random is the default
  frac: 0.1
  random_state: 42
  group_name: data_exploration
```

### Fixed Count

```yaml
type: dagster_component_templates.SampleComponent
attributes:
  asset_name: events_1000
  upstream_asset_key: raw_events
  method: random
  sample_size: 1000
  random_state: 7
  group_name: dev_samples
```

### First N / Last N

```yaml
type: dagster_component_templates.SampleComponent
attributes:
  asset_name: first_100_orders
  upstream_asset_key: raw_orders
  method: head             # or 'tail' for the last N
  sample_size: 100
```

### Every Nth Row

```yaml
type: dagster_component_templates.SampleComponent
attributes:
  asset_name: every_tenth_event
  upstream_asset_key: raw_events
  method: every_nth
  sample_size: 10          # step value — keeps rows 0, 10, 20, ...
```

### Skip First N

```yaml
type: dagster_component_templates.SampleComponent
attributes:
  asset_name: events_minus_preamble
  upstream_asset_key: raw_events
  method: skip_head
  sample_size: 4           # drop the first 4 header rows
```

## Inputs and Outputs

| | Type | Description |
|-|------|-------------|
| Input | `pd.DataFrame` | The upstream DataFrame to sample from |
| Output | `pd.DataFrame` | Sampled DataFrame with reset integer index |

## IO Manager Note

This component returns a plain `pd.DataFrame`. Ensure your Dagster environment
has an IO manager configured that can handle DataFrames.

## Notes

- For `method=random`: exactly one of `sample_size` or `frac` must be specified.
- For `method` in `head` / `tail` / `every_nth` / `skip_head`: `sample_size` >= 1 is required; `frac` / `random_state` / `replace` / `weights` are ignored.
- The output index is always reset to a clean integer range.
- `weights` must refer to a numeric column in the upstream DataFrame (random only).
- Use `replace=True` with `frac > 1.0` for bootstrap samples larger than the input (random only).
