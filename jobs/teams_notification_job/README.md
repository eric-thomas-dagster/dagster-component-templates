# TeamsNotificationJobComponent

Send a message to a Microsoft Teams channel via incoming webhook.

## Dependencies
- `requests`

## What this is
A **Dagster job** (no asset materialized) — the component installs a `dg.job`
plus an optional schedule. Lineage graph stays clean; the work happens
behind the scenes.
