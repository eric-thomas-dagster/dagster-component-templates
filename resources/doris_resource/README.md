# doris_resource

## Purpose

Provides an [Apache Doris](https://doris.apache.org/) connection resource — SQLAlchemy URL (via the MySQL wire protocol on port 9030) + a raw PyMySQL connection + the HTTP frontend endpoint (port 8030) for Stream Load / admin REST calls.

Doris is an MPP real-time OLAP database (similar to ClickHouse / Druid / StarRocks). It speaks the MySQL protocol on its query port, so generic SQL components in this registry work as-is once you point them at this resource's `connection_string`.
