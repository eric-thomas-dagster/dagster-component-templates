# RssFeedSensor

Polls a public RSS or Atom feed and triggers a RunRequest for each new entry seen since the last evaluation (by entry GUID). Useful for newswire / blog / regulatory-filing-driven pipelines.

## Fields

### Required

| Field | Type | Description |
|---|---|---|
| `sensor_name` | `str` | Unique sensor name. |
| `asset_keys` | `List[str]` | Slash-separated asset keys to materialize when an event matches. |
| `feed_url` | `str` | RSS or Atom feed URL. |

### Sensor configuration

| Field | Type | Default | Description |
|---|---|---|---|
| `minimum_interval_seconds` | `int` | `60` | Minimum seconds between sensor evaluations. |
| `default_status` | `str` | `"STOPPED"` | 'RUNNING' or 'STOPPED'. |

### Other
