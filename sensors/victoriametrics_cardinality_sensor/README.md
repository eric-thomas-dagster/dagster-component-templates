# victoriametrics_cardinality_sensor

Watch [VictoriaMetrics](https://victoriametrics.com/) series cardinality via `/api/v1/status/tsdb`. When total active series crosses a configurable threshold, emit a `RunRequest` against a remediation job ŌĆö or, if no target job is set, surface a warning `SkipReason` (UI-visible) so the operator notices.

VM (like Prometheus) performs poorly past ~10M active series. Runaway cardinality is usually a misnamed label ŌĆö `user_id` as a tag, `request_id` as a tag, a label that should have been a field. This sensor surfaces the spike *before* query latency tanks.

## Example ŌĆö pure warning mode (no target job)

```yaml
type: dagster_component_templates.VictoriaMetricsCardinalitySensorComponent
attributes:
  sensor_name: vm_high_cardinality_watch
  base_url_env_var: VM_BASE_URL
  threshold_series: 1000000     # warn past 1M series
  interval_seconds: 300         # poll every 5 min
  top_n_metrics: 10
```

The sensor stays in **SkipReason mode** ŌĆö every evaluation logs the current cardinality + delta-vs-previous + top-5 metrics by cardinality, prefixed with `OK below threshold` or `WARN: cardinality Ōēź threshold`. Useful as a always-on observability surface without forcing a remediation flow.

## Example ŌĆö alert with remediation job

```yaml
type: dagster_component_templates.VictoriaMetricsCardinalitySensorComponent
attributes:
  sensor_name: vm_cardinality_alert
  base_url_env_var: VM_BASE_URL
  threshold_series: 5000000
  target_job_name: my_cardinality_drop_job   # user-defined Dagster job
  target_job_tags:
    severity: high
  interval_seconds: 60
```

On breach, the sensor emits a `RunRequest` with cardinality detail in tags (`vm/cardinality_total_series`, `vm/cardinality_total_label_pairs`). The user's `my_cardinality_drop_job` reads those tags and runs the remediation (drop a series, alert PagerDuty, etc.).

## Cursor / delta tracking

The sensor persists the last-seen `totalSeries` value in its cursor so each evaluation can log `╬ö=+12345 vs last poll`. Useful for catching gradual growth before it crosses the threshold.
